About

{{ settings.title }} helps you understand how organisations use data about you. It guides you through making requests that change how it's used.

We're currently focussing on covering fintech organisations. We'll be expanding our coverage in the future.

For developers

This website uses a new and open dataset of machine-readable privacy notices. Developers can use our API to integrate this dataset with their application.

Learn how to use the API on GitHub

Contribute

We’re looking for partners to expand the organisations we cover. You can help by converting a text privacy notice into a machine-readable privacy notice, or by contributing your organisation's privacy notice.

Start a contribution with the form or request an organisation by email
